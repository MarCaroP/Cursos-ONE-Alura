****Lo que aprendimos en esta aula:
FOOTER
- Propiedad width para aumentar el espacio de nuestro elemento || usuamos porcentajes para ocupar el 100% del conteiner, distribuyendo los elementos y usando un margen entre ellos para completar el conteiner 
- El space-around || se usa cuando necesitamos espacio delante, despues del elemento
- Propiedad flex-direction || se usar para que los elementos se pongan en otra direccion, distinta de un lado al otro. puede ser columns o row entre otras
- Propiedad height para setear la altura de nuestro elemento || limita las columnas a un tamaño especificio
- Propiedad flex-wrap || quiebra y sigue a la columna siguiente, se completa hasta los pixeles indicados y sigue, se usa para listar. 
JUSTIFY-CONTENT 
 Podemos distribuir los elementos dentro del padre de varias maneras, podemos por ejemplo:
 Poner todo el espacio a la izquierda, lanzando el contenido a la derecha con justify-content: flex-end.
 Poner todo el espacio a la derecha, lanzando el contenido a la izquierda con justify-content: flex-start (que es el modelo).
 Poner todo el espacio a la izquierda y a la derecha, lanzando el contenido al centro con justify-content: center.
 Poner todo el espacio entre los elementos como vimos antes con justify-content: space-between.
 Y una posibilidad muy interesante es también poner espacio alrededor de los elementos. Para ello podemos utilizarjustify-content: space-around.
